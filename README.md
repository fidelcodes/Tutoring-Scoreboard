# Tutoring-Scoreboard: Player Score Manager

A simple, modular Java console application designed for managing and tracking individual player scores, complete with dynamic ranking and persistent data storage.

This project was developed for practical use in educational settings but is applicable for any scenario requiring lightweight console-based data management and visualization.

## Project Highlights

### Tech Stack
* **Language:** Java
* **Core Concepts:** Object-Oriented Programming (OOP), File I/O, Sorting Algorithms

### Key Features
* **Persistent Data Storage:** Implements **File I/O** to reliably save and load player data (scores and names), ensuring that the scoreboard state is maintained across different sessions.
* **Dynamic Leaderboard:** Utilizes an efficient **sorting algorithm** to dynamically rank players and display a score visualization.
* **CRUD Operations:** Provides core console functionality to add new players, delete existing records, and view the updated scoreboard.
* **Robust Input Validation:** Includes multiple checks for user input errors to ensure program stability and a smooth user experience.
* **Modular Design:** Code is separated into logical classes and methods, adhering to **OOP principles** for improved readability, maintainability, and scalability.
